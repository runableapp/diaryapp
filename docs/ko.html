Diary — 일기

아름답고 단순하며 안전한 개인 일기 앱입니다. 자유롭게 글을 쓰고 사진을 첨부하세요. 비밀번호 보호와 AES-256 암호화로 소중한 기록을 안전하게 지켜줍니다.

Linux Windows macOS

기능

필요한 것은 모두, 불필요한 것은 없이

Diary는 단순함에 집중합니다. 달력, 리치 텍스트 에디터, 강력한 암호화. 클라우드도, 계정도 필요 없습니다. 당신의 기록은 당신의 기기에만 저장됩니다.

🔒

비밀번호 보호

마스터 비밀번호로 일기를 잠글 수 있습니다. 설정한 비활성 시간 이후 자동 잠금이 동작합니다.

🗓

달력 탐색

직관적인 월간 달력으로 날짜별 항목을 쉽게 찾을 수 있습니다. 점 표시로 기록이 있는 날짜를 한눈에 확인합니다.

✏

리치 텍스트 에디터

굵게, 기울임, 목록, 제목, 색상 등 다양한 서식을 지원합니다. 완전한 툴바로 자유롭게 작성하세요.

📷

사진 첨부

이미지를 일기 본문에 바로 삽입할 수 있습니다. 여러 장의 사진으로 하루의 순간을 생생하게 남기세요.

🔍

전체 텍스트 검색

내용, 제목, 날짜로 전체 기록을 즉시 검색합니다. 원하는 기억을 몇 초 만에 찾을 수 있습니다.

🌐

다국어 지원

영어, 한국어, 일본어, 중국어, 프랑스어, 독일어 등 14개 언어를 지원합니다. 날짜 표시는 시스템 로캘에 맞춰집니다.

🛡

AES-256 암호화

기록은 AES-256으로 암호화됩니다. 데이터는 로컬에만 저장되며 클라우드, 추적, 텔레메트리가 없습니다.

🎨

라이트/다크 테마

원하는 분위기에 맞게 테마를 선택하세요. 설정 화면에서 라이트/다크 모드를 전환할 수 있습니다.

💻

크로스 플랫폼

Linux, Windows, macOS에서 네이티브로 실행됩니다. Flutter 기반의 일관된 사용 경험을 제공합니다.

스크린샷

실제 화면으로 확인하세요

잠금 화면부터 에디터, 검색, 설정까지 앱의 주요 흐름을 빠르게 살펴볼 수 있습니다.

안전한 잠금 화면

깔끔한 잠금 화면에서 시작합니다. 마스터 비밀번호를 입력해 기록에 접근하세요. 일정 시간 미사용 시 자동 잠금으로 프라이버시를 보호합니다.

하루를 기록하세요

달력에서 날짜를 선택하고 바로 작성을 시작하세요. 서식, 제목, 목록 등을 지원하며 입력 중 자동 저장됩니다.

풍부한 서식

굵은 제목, 컬러 텍스트, 번호/글머리 목록 등으로 표현력을 높이세요. 필요한 모든 서식 기능을 툴바에서 바로 사용할 수 있습니다.

사진으로 남기는 기억

사진을 여러 장 첨부해 텍스트와 함께 기록할 수 있습니다. 소중한 순간을 더 생생하게 보관하세요.

검색과 탐색

키워드, 제목, 날짜로 모든 항목을 검색합니다. 목록 보기에서 날짜/제목/내용 미리보기를 한눈에 확인할 수 있습니다.

설정 및 보안

비밀번호, 잠금 시간, 데이터 폴더, 테마를 손쉽게 설정하세요. 누구나 쉽게 사용할 수 있도록 설계했습니다.

안전한 삭제

항목 삭제 시 확인 절차를 거칩니다. 실수로 삭제되는 일을 줄여 소중한 기록을 보호합니다.
